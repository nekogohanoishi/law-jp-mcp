## 1. 再審開始決定に対する即時抗告棄却決定に対する特別抗告事件（hanrei-auto-95593）
**最二小決 令和8年2月24日 / 令和5(し)155**

### 事件：滋賀県「日野町事件」の再審
- **1984年**、滋賀県日野町で酒店経営の女性が強盗殺人された事件
- 常連客の**阪原弘さん**が「自白」に基づき逮捕、無期懲役が確定（2000年）
- 阪原さんは一貫して無実を主張し続けたが、**2011年に服役中に75歳で病死**
- 遺族が第2次再審請求 → 大津地裁（2018年）・大阪高裁（2023年）が再審開始を支持 → 最高裁も棄却で確定

### 法的意義
- **戦後初の「死後再審」確定** — 本人死亡後でも冤罪の名誉回復が可能であることを示した
- 決定的証拠は**警察が送致していなかった写真ネガ**。捜査報告書の写真が入れ替えられていたことが判明
- 第1次再審請求から**約25年**かかっており、検察の抗告権制限の議論を加速させた
- 日弁連が会長声明で高く評価、検察の不服申立て禁止を求めている

### 社会的反響
- 時事・日経・東京新聞・毎日など主要メディアが一斉に報道
- 遺族の長男「父ちゃんやったね」という声が話題に
- 刑事訴訟法改正（証拠開示制度・検察の抗告権）の国会審議に直接影響

## 2. 保有個人情報不開示決定処分取消請求事件（hanrei-auto-95570）
**最三小判 令和8年2月20日 / 令和6(行ヒ)362**

### 事件の背景
- 刑務所で収容中に**同室者からいじめを受けて死亡した母**の調査記録について、遺族（子）が個人情報保護法に基づき開示請求
- 行政庁が不開示決定 → 遺族が取消訴訟を提起

### 法的意義
- **死者に関する情報が遺族を「本人」とする保有個人情報に当たるか** → **当たらない**と最高裁が判断
- 平成31年最判（故人の印鑑届書事件）の「個別判断説」を**行政機関の保有情報（刑事施設の調査記録）**にまで拡張適用
- 現行法が**死者の情報に対する遺族の開示請求権を保障していない**ことが改めて確認された
- 補足意見（林・石兼裁判官）で、業務災害の保険給付や相続税調査など**遺族自身が法的地位を取得している場合は別**であることを注記

### 社会的意義
- 刑務所内のいじめという**人権侵害の調査記録**へのアクセスが遺族に認められなかった点で、被収容者の人権保護・施設内の透明性の議論に影響
- 学術界では「死後の個人情報保護」に関する**立法的手当の必要性**が指摘されている
- メディアの大規模報道は現時点では確認されず、今後判例評釈が出る見込み
